Este aplicativo foi desenvolvido para auxiliar os alunos da faculdade a calcularem suas notas bimestrais e final.

Nesta faculdade se aplica um cálculo de notas ponderadas para se encontrar a média para cada bimestre e a nota de aprovação:

MB1 = NM1 * 0,4 + NB1 * 0,6
MB2 = NM2 * 0,4 + NB2 * 0,6
MF = (MB1 * 2 + MB2 * 3)/5

Legenda:

Nota Mensal 1 = NM1
Nota Bimestral 1 = NB1
Média Bismestral 1 = MB1
Nota Mensal 2 = NM2
Nota Bimestral 2 = NB2
Média Bismestral 2 = MB2
Média Final = MF
